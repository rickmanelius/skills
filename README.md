# Save Your Startup — Claude Code Skills

Claude Code skills based on [*Save Your Startup*](https://www.amazon.com/Save-Your-Startup-Business-Rescuing-ebook/dp/B0D32VDYPS) by Rick Manelius.

## Installation

In Claude Code:

```
/plugin marketplace add rickmanelius/skills
/plugin install save-your-startup
```

That's it — Claude Code will fetch the repo and register all 12 skills automatically.

## Skills

| Skill | Command | When to use |
|-------|---------|-------------|
| **Paint Done** | `/sys:paint-done` | Define "done" before any work begins |
| **Chunk** | `/sys:chunk` | Break a 10-year vision into today's next action |
| **High Medium Low** | `/sys:high-medium-low` | Define scope tiers so you build the right amount |
| **Assumptions Audit** | `/sys:assumptions-audit` | Surface hidden cost and revenue assumptions |
| **Napkin Math** | `/sys:napkin-math` | Walk backward from a revenue target to daily growth |
| **Capability Willingness Matrix** | `/sys:capability-willingness-matrix` | Assess if you have the skills AND willingness |
| **Founder Type Check** | `/sys:founder-type-check` | Honest check — delusional, pretender, or real deal |
| **Fear Setting** | `/sys:fear-setting` | Name fears, write mitigations, rate severity |
| **WIIFM Check** | `/sys:wiifm-check` | 7 questions every prospect silently asks |
| **Five Finger Pitch** | `/sys:five-finger-pitch` | Build a 5-finger, 5-minute verbal pitch |
| **Ruthless Elimination** | `/sys:ruthless-elimination` | Reclaim 5-15 hours/week with Delete, Delegate, Delay |
| **Break Bottleneck** | `/sys:break-bottleneck` | Find and fix your #1 throughput constraint |

Over 50 candidate skills were identified from the book — these 12 scored highest on universality, power, and AI-transferability. See the [full ranking](context/analysis/00-master-ranking.md) and [file an issue](https://github.com/rickmanelius/skills/issues) if you'd like another one included.
